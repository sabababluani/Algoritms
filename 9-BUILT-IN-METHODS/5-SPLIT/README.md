# String Split (Custom Implementation)

## Description

This project implements a custom version of the native JavaScript `String.prototype.split()` method.

The goal is to create a function named `stringSplit` that mimics the behavior of the built-in `split()` method.

---

## Function Signature

```javascript
function stringSplit(string, separator) {}
```

    Parameters:

        string (String): The string to be split.

        separator (String): The delimiter used to determine where to split the string.

    Returns:

        An array of substrings split based on the specified separator.
